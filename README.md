# Autonomous Line Following Robot

## Project Overview

This project demonstrates an autonomous robot capable of line following and obstacle detection using Arduino.

The robot uses multiple sensors and control algorithms to navigate a predefined path while avoiding obstacles.

---

## Technologies Used

Arduino Uno  
IR Sensors  
Ultrasonic Sensor  
L298N Motor Driver  
DC Motors  

---

## System Architecture

Sensor Input → Arduino Controller → Motor Driver → Robot Movement

---

## Features

Line following using IR sensors  
Obstacle detection using ultrasonic sensor  
Real-time motor control  

---

## Applications

Industrial mobile robots  
Warehouse automation  
Autonomous navigation systems
